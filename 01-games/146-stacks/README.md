# Stacks

> **Category:** Education | **Format:** Interactive Quiz

An educational activity for young learners exploring **Stacks** through multiple-choice questions with instant feedback.

## Features

- 10 curated questions with 4 answer choices
- 8 visual themes (Terminal, Midnight, Ember, Arctic, Phosphor, Signal, Redshift, Sand)
- 3 languages (English, French, Arabic with RTL)
- High score tracking with localStorage
- Timer and activity log
- Sound effects (Web Audio API)
- Fully offline — single HTML file, no dependencies

## How to Play

1. Open `index.html` in any browser
2. Choose your theme and language
3. Answer questions and beat your high score!
